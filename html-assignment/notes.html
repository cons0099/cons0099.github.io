HTML Tags

Emphasis
<em></em> a string of emphasized, slightly more important text,screen readers will change their voice for this text.
Abbreviation
<abbr title="..."></abbr> and acronym or abbreviation, like "HTML","CSS", etc. Title attribute contains the expanded version, like <abbr title="HyperText Markup Language">HTML</abbr>.
Citation
<cite></cite> a citation from another source, often used with quotations. A person's name, URL, book, movie, etc.
Deleted Content
<del datetime="..."></del> content that was deleted after the document was published. Datetime defines when thr content was removed.
Highlighted Text
<mark></mark> used to a piece of text for reference. The keywords in a search results page, the current navigation item, etc.
